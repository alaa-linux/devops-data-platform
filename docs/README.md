# Documentation

Ce dossier contient toute la documentation technique du projet.

## Structure

- `architecture/` : schémas, décisions et documentation d’architecture
- `procedures/` : procédures techniques reproductibles
- `recaps/` : récapitulatifs de progression
- `screenshots/` : captures d’écran du projet
- `troubleshooting/` : problèmes rencontrés et corrections

## Objectif

Centraliser toute la documentation afin de rendre le projet reproductible, maintenable et présentable dans un portfolio professionnel.
