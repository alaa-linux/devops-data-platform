# Airflow

Ce dossier contient l’orchestration des pipelines du projet.

## Technologies prévues

- Apache Airflow
- DAGs Python
- Orchestration ETL

## Structure

- `dags/` : pipelines Airflow
- `logs/` : logs Airflow
- `config/` : configuration
- `plugins/` : plugins et operators personnalisés

## Objectif

Automatiser et orchestrer les pipelines Data Engineering, Spark, Kafka et Machine Learning.
