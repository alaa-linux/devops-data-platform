# Warehouse

Ce dossier contient la couche Data Warehouse du projet.

## Technologies prévues

- PostgreSQL
- SQL Analytics
- ETL SQL

## Structure

- `schema/` : schémas SQL
- `etl/` : scripts ETL SQL
- `queries/` : requêtes analytiques

## Objectif

Centraliser les données nettoyées et structurées pour l’analyse, le reporting et le Machine Learning.
