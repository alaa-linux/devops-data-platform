# Ingestion

Ce dossier contient les composants d’ingestion de données du projet.

## Sources prévues

- API
- CSV
- logs système

## Structure

- `api/` : extraction depuis des APIs
- `csv/` : ingestion de fichiers CSV
- `logs/` : ingestion de logs système/applicatifs

## Objectif

Centraliser les données brutes avant leur stockage dans le Data Lake.
